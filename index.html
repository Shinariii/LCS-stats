League of Legends Esports Data Analysis & Predictive Modeling

This project delves into the competitive world of League of Legends esports, utilizing match data from the 2025 and 2026 seasons. The goal is to perform a comprehensive data analysis, engineer new features to quantify player and team performance, and build a predictive model to forecast game outcomes. This notebook showcases skills in data manipulation, SQL querying, advanced feature engineering, data visualization, and machine learning, making it a robust portfolio piece for aspiring data professionals.

Key Achievements:

  • Data Integration: Combined and managed large datasets efficiently using SQLite.
  • Feature Engineering: Created five impactful features that capture critical aspects of player and team performance: gamelength_minutes, tkpm, golddiff_gpm (refined to golddiffat10), gold_to_dmg_ratio, and kills_to_obj_ratio.
  • Visualization: Developed clear and actionable visualizations, including focused analyses on the LCS 2026 season for team objective control and player early-game gold differential.
  • Predictive Modeling: Built a baseline Logistic Regression model to predict game outcomes, achieving an accuracy of 0.9242 and an ROC AUC of 0.9890.
